# To To Bot

To To Bot is a discord bot made with [DISCORD.JS](https://github.com/discordjs/discord.js)

Features 0.1:

  - Commands [aliases]
    --addchannel
    --clear
    --deletechannel
    --die [d,dice]
    --help
    --info
    --play
    --prefix
    --queue
    --skip
    --stop
 

### Installation

To To requires [Node.js](https://nodejs.org/) v4+ to run.

Install the dependencies and devDependencies and start the server.

```sh
$ cd totoBot
$ npm install -d
$ node app
```
